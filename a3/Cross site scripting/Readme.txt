Το πρόβλημα βρίσκεται στις φόρμες που γίνονται POST από τους χρήστες και δεν ελέγχονται 
στην συνέχεια. Αν σε μία τέτοια φόρμα ο χρήστης προσθέσει javascript κώδικα,και δεν ελεγχθεί σωστά,
τότε με το που γίνει το submit ο κώδικας θα τρέξει κανονικά μαζί με αυτόν της σελίδας, για όποιον επισκεφθεί
μετά την σελίδα. Ο κίνδυνος τώρα ελαχιστοποιείται όταν σε κάθε φόρμα συμπλήρωσης πρίν να διαβαστούν τα δεδομένα
γίνει μία διαδικασία φιλτραρίσματος της εισόδου. Αυτό γίνεται με το να ψάχνεται η είσοδος και να αντικαθίστανται 
τα σύμβολα '<','>','&', '\' με &lt, &gt, &amp, &quot αντίστοιχα αλλάζοντας την μορφή της ώστε να μην μπορεί να εκτελεστεί 
πια ο javascript κώδικας.



public String clean(String input){
        
        input = input.replaceAll("<", "& lt;").replaceAll(">", "& gt;");
        
        input.replaceAll("script", "");
        
        input = input.replaceAll("&", "&amp");
        
        input.replaceAll("\"","&quot");
        
        return input;
        
    }
